**EasyParkPlus Parking Manager with EV Charge Management**

Implementation of multi-city, multi-site and multi-level parking lot facility with EV charge management

**\## Prerequisites**

- **Python 3.11+**

*Run the application*
python ParkingManager.py
•	This will launch the User Interface.
**Functionalities**
*Parking Lot Creation*
1.	Select City.
2.	Enter the Site Name.
3.	Enter the Level Number.
4.	Enter the parking lot attributes:
      o	Number of Regular Slots
      o	Number of EV Slots
      o	Number of EV Chargers
5.	Click Create Parking Lot button.
*Parking a Vehicle*
1.	Select City, Site, and Level from the Car Management section.
2.	Enter the vehicle details.
3.	Click Park Vehicle button.
&Removing a Vehicle*
1.	Select City, Site, and Level from the Car Management section.
2.	Enter the Slot Number.
3.	Check the EV checkbox if removing an electric vehicle.
4.	Click Remove Vehicle button.
*Display Parking Lot Status*
•	Click the Current Lot Status button.
*Display EV Charge Status*
•	Click the EV Charge Status button.
*Searching Vehicles*
1.	Select City, Site, and Level from the Car Management section.
2.	Select the search option from the dropdown menu.
3.	Enter the value you are searching for.
4.	Click Search button.
